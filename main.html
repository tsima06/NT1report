Report: ISO/OSI Model and Protocol Assignment

1. ISO/OSI Model Description

The OSI (Open Systems Interconnection) model defines seven layers that describe network communication:

  1. Physical — transmission of bits via media (RS-232, USB, IEEE 1394)
  2. Data Link — framing, MAC addressing (Ethernet, ARP, Wi-Fi)
  3. Network — logical addressing, routing (IPv4, IPv6, ICMP, IGMP)
  4. Transport — end-to-end delivery (TCP, UDP)
  5. Session — connection management (NetBIOS, SIP)
  6. Presentation — data format, encryption (MIME, SSL, TLS)
  7. Application — user-level services (HTTP, FTP, DNS, DHCP, Telnet)
